Servidor HTTP

O servidor HTTP é a ferramenta responsável pela distribuição de documentos HTML atráves do protocolo HTTP em um sistema de arquivos.

Servidor HTTP

Este Servidor HTTP pode ser configurado para ser acessado apenas em uma rede interna (Intranet) ou uma rede externa (Internet).

Apache Web Server

O Apache Web Server é um Servidor Web, open source, multiplatafor mais utilizado em todo mundo.

Apache Web Server

As principais características:

  • Fexível
  • Altamente configurável
  • Robustez
  • Escalabilidade
  • Pode ser configurado para diferentes funções
  • É composto de módulos separados onde cada um implementa uma característica diferente.

Instalação no Linux

Atualize a lista do apt

 $ sudo apt-get update 
 $ sudo apt-get upgrade

Instale o Apache e o PHP

 $ sudo apt-get install apache2 php5

Instalação no Linux

Instale o MySQL

 $ sudo apt-get install mysql-server-5.5 php5-mysql

Reinicie o Apache

 $ sudo /etc/init.d/apache2 restart

Arquivos de configuração

Os arquivos de configuração estão dispostos em:

Linux: /etc/apache/conf.httpd.conf

WAMP: wamp/bin/apache/apache2.4.9/conf/conf.httpd.conf

Iniciando/Desligando
o Servidor

Para o servidor no Linux

#Parar
$ sudo /etc/init.d/apache2 stop

#Iniciar
$ sudo /etc/init.d/apache2 start

#Reiniciar
$ sudo /etc/init.d/apache2 restart
